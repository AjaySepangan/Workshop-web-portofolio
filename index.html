Tentang Saya

Saya adalah seorang Front-End Developer dengan minat besar dalam desain UI/UX yang modern, minimalis, dan berfokus pada pengalaman pengguna. Saya percaya bahwa antarmuka yang baik tidak hanya indah secara visual, tetapi juga intuitif dan fungsional. Dengan latar belakang dalam pengembangan web responsif dan interaktif, saya senang mengubah ide menjadi tampilan digital yang menarik menggunakan teknologi seperti HTML, CSS, JavaScript, serta framework modern seperti React atau Tailwind CSS. Saya selalu terbuka untuk tantangan baru dan terus belajar untuk menciptakan solusi digital yang lebih baik.

Projek

Website Portfolio Pribadi

Sebuah website portofolio yang dirancang dengan pendekatan modern dan minimalis, menggunakan kombinasi warna emas dan hitam untuk memberikan kesan elegan. Dibangun dengan HTML, CSS, dan JavaScript, website ini sepenuhnya responsif dan memuat informasi tentang profil, projek, serta form kontak yang berfungsi. Desain fokus pada user experience (UX) agar pengunjung mudah menavigasi dan menemukan informasi penting.

Aplikasi To-Do List Sederhana

Aplikasi manajemen tugas berbasis web dengan fitur tambah, hapus, dan tandai tugas selesai. Dibangun menggunakan JavaScript murni (vanilla JS), aplikasi ini memiliki antarmuka yang bersih dan ringan. Didesain untuk digunakan pada perangkat mobile maupun desktop, dengan struktur kode yang mudah dikembangkan lebih lanjut.
